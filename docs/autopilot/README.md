Tento priecinok je jadro systemu pre bezpecny autonomny rozvoj KomArena.sk. Autopilot pracuje cez samostatne vetvy, Pull Requesty, backlog, QA, dizajn, SEO a stavovy subor.

## Rychly start

1. Precitaj AGENTS.md.
2. Precitaj docs/autopilot/state.md.
3. Precitaj docs/autopilot/backlog.md.
4. Vyhodnot ulohu podla docs/autopilot/decision-matrix.md.
5. Over schvalovaciu uroven v docs/autopilot/approval-gates.md.
6. Uprav iba povolene subory.
7. Spusti kontroly.
8. Vytvor PR a aktualizuj state a decision log.
