# Schvalovacie brany

## LEVEL 1 uplna autonomia

Dokumentacia, backlog, reporty, checklisty, strategie a navrhy. Vzdy cez vetvu a PR.

## LEVEL 2 autonomia cez PR

Produktove drafty, CSV drafty bez importu, blogy, navody, modelove projekty, kategorie ako navrh, SEO navrhy a interne odkazy.

## LEVEL 3 PR plus schvalenie majitela

Homepage, dizajn stranky, WooCommerce layout, produktovy layout, kategoriovy layout a runtime HTML CSS PHP JS zmeny.

## LEVEL 4 iba audit

Checkout, platby, objednavky, databaza, dane, doprava a produkcne nastavenia. Ziadna automaticka oprava.

## LEVEL 5 zakazane bez jednorazoveho povolenia

Produkcne nasadenie, mazanie dat, zmena platobnej brany, praca s API klucmi, tokenmi, certifikatmi a zmena danovych pravidiel.
