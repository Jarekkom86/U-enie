# Category Engine

Category Engine udrziava kategorie tak, aby neboli iba zoznamom produktov, ale predajnou a edukacnou landing page.

## Kategoria ma obsahovat

- kratke vysvetlenie
- komu je urcena
- ako vybrat produkt
- odporucane clustre
- suvisiace navody
- suvisiace projekty
- bezpecnostne upozornenia
- interne odkazy

Textovy navrh je LEVEL 2. Runtime layout zmena je LEVEL 3.
