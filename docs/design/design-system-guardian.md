# Design System Guardian

Design System Guardian strazi, aby produkty, blogy, kategorie, homepage navrhy, CTA, karty a tabulky drzali jednotny styl KomArena.

## Kontroluje

- farby z design tokens
- jednotne karty
- jasne CTA
- citatelne tabulky
- mobilne zobrazenie
- warning bloky
- technicky a doveryhodny ton

Runtime dizajnove zmeny su LEVEL 3 a vyzaduju schvalenie majitela.
