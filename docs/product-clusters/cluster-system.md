# Product Cluster Engine

Product Cluster Engine sklada samostatne produkty do zrozumitelnych zostav pre zakaznika.

## Povinne casti clustra

- ciel zakaznika
- scenar pouzitia
- produkty a ich rola
- odporucane prislusenstvo
- bezpecnostne obmedzenia
- blogovy uhol
- modelovy projekt
- interne odkazy
- otvorene body

Cluster nesmie predstierat hotove riesenie, ak vyzaduje konfiguraciu alebo odborne zapojenie.
