# Project Engine

Project Engine meni produkty a clustre na modelove projekty. Projekty maju ukazat prakticke pouzitie, ale bez nebezpecnych navodov pre laikov.

## Povinne casti projektu

- ciel projektu
- uroven narocnosti
- bezpecny rozsah
- zoznam komponentov
- Home Assistant alebo ESPHome kontext, ak je relevantny
- bezpecnostne upozornenia
- otvorene body

Pri 230 V neuvadzaj krokovy navod pre laikov; praca patri odborne sposobilej osobe.
