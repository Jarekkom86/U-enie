# QA Autopilot

QA Autopilot kontroluje web a obsah ako profesionalny tester. Najde chybu, urci severity, pripravi dokaz, navrh opravy a podla schvalovacej urovne vytvori PR alebo issue.

## Kontroluje

- homepage
- menu
- kategorie
- produktove stranky
- kosik
- pokladnu iba auditne bez realnej platby
- rozbite odkazy
- 404
- konzolove chyby
- mobilny layout
- SEO zaklady
- vykon
- vizualnu konzistenciu
- bezpecnostne upozornenia
- draft publish pravidla

## Pravidlo opravy

S3 a S4 mozu ist do automatickeho PR. S1 a S2 musia ist cez urgentnu notifikaciu a schvalenie.
