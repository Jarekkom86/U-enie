# Severity System

| Severity | Definicia | Priklady | Akcia |
|---|---|---|---|
| S1 | Kriticke ohrozenie webu, platieb, checkoutu, bezpecnosti alebo databazy. | web nejde, platba zlyhava, unik dat | urgent issue, ziadny autofix |
| S2 | Vysoky dopad na nakupnu cestu alebo produkt. | produkt nejde kupit, kosik pada | issue a navrh opravy |
| S3 | SEO, obsah, interne odkazy alebo slabsi popis. | chyba meta, zly odkaz | automaticky PR povoleny |
| S4 | Preklep, formatovanie, dokumentacia. | typo, checklist | automaticky PR povoleny |

Checkout, platby, objednavky, databaza, dane a doprava su vzdy audit-only.
