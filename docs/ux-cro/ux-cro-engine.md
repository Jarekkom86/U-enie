# UX CRO Engine

UX CRO Engine znizuje neistotu zakaznika a zlepsuje nakupnu cestu.

## Kontroluje

- hodnotovu ponuku
- CTA
- doveryhodnost
- produktove benefity
- odporucane prislusenstvo
- citatelnost
- mobilne zobrazenie
- kategoriovu orientaciu

Dokumentacne navrhy su LEVEL 1 alebo LEVEL 2. Runtime layout zmeny su LEVEL 3.
